# COD4-Online-Patch
This is a patch is made to fix errors like (Key Code in Use) In online plays for Call of Duty 4: Modern Warfare. This Patch guarantees that You could play online or create private server without problems.

This Patch is for the latest version "1.7" And it works on steam and cracked versions. It's not guaranteed that it will work on older versions


# Installation
It's very simple just navigate to the COD4 Directory (typically `C:\Program Files (x86)\Steam\steamapps\common\Call of Duty 4`) then copy the downloaded file inside and you are good to go.

**Notes:**
1- BunkerBuster must be installed and set-up for most of the games to work especially Gameranger ones
2- In case using private match. All of the players must apply the patch or they will get "Key code in use" error
